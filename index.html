WIN10Alt Tab卡顿联想电脑解决办法

导火索

今天自用联想Y7000P出现问题，问题也不大，就是卡了，

  • i7-9750
  • 1T三星纯固态
  • 1660Ti
  • 16G内存

至于怎样的卡，比如按下WIN+R卡，用输入法打字卡，切换窗口卡

解决办法

  • 昨天所做的努力

    • 一直在尝试，我打开程序和功能，查看那些程序在最新的安装日期，查看最近更新，看看WINDOWS有没有更新最新的补丁，找到一个是10月30号的 但是已经固化了 不敢轻易卸载，听说之前有百度吧友卸载固化补丁结果蓝屏了，只能继续上网找方法。

  • 今天所得的方法

    • 右键 我的电脑- 管理 - 服务和应用程序- 服务。找到 Lenovo System Interface Foundation 服务禁用掉重新启动即可（不重启也没事，立马见效）
    • 貌似这是一个联想支持服务，关了之后MCAFREE好像变得不正常了 不过没事 我认为有一个微软安全中心足够了。

  • 如何卸载Lenovo System Interface Foundation

    • 在“设备管理器”中，展开“系统设备”类别，右键单击“System Interface Foundation V2 Device”，然后选择“卸载”。

    • 从管理员命令提示符处，执行以下命令：“c：\ windows \ system32 \ imcontroller.infinstaller.exe -uninstall”
